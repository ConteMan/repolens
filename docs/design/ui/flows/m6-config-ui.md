# M6 主路径与状态矩阵

```mermaid
flowchart TD
  A[输入本地 Git 工作树绝对路径] --> B{工作树可读取?}
  B -- 否 --> C[路径字段错误]
  C --> A
  B -- 是 --> D[读取仓库 document 与有效默认值]
  D --> E[编辑仓库域字段和规则]
  E --> F[服务端校验]
  F --> G{有字段错误?}
  G -- 是 --> H[字段内错误摘要]
  H --> E
  G -- 否 --> I[写入确认: 目标路径和 YAML Diff]
  I --> J{revision 一致且确认?}
  J -- 磁盘已变更 --> K[冲突: 重新读取]
  K --> D
  J -- 取消 --> E
  J -- 是 --> L[原子写入]
  L --> M{选择构建输出}
  M -- 缓存目录 --> N[工作树构建到缓存根]
  M -- 自定义目录 --> Q{路径安全且可覆盖?}
  Q -- 否 --> R[路径字段错误或覆盖确认]
  R --> M
  Q -- 是 --> N
  N --> S{构建成功?}
  S -- 是 --> O[统计、警告、最终输出路径]
  S -- 否 --> P[失败阶段、错误、最近成功结果]
```

| 环节 | 正常 | 异常/限制 | 用户下一步 |
|---|---|---|---|
| 选择 | 显示绝对路径输入 | 相对路径、非目录、工作树不可读 | 修改路径 |
| 编辑 | 显示来源与当前值 | YAML 解析失败、字段无效 | 查看行内错误，修正或保留原文件 |
| Diff | 仅显示受控字段变动 | 无变更 | 返回编辑，不显示写入按钮 |
| 写入 | 记录已写入路径 | token/权限/并发变更失败 | 重新读取，避免覆盖未知修改 |
| 输出 | 默认缓存；可输入会话级自定义绝对目录 | 危险路径、仓库关系、未知非空目录、已有 repolens 输出 | 修改路径或明确确认覆盖 |
| 构建 | 阶段、Stats、Warnings、最终路径 | source/config/theme/site/发布失败 | 查看失败阶段，返回配置或重试 |
